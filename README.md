# Banking-Analysis-Dashboard
Developed an interactive Power BI Banking Analysis Dashboard to analyze key financial metrics such as loans, deposits, revenue, and customer performance across different bank branches. The dashboard provides year-wise trend analysis, branch-wise financial insights, and customer segmentation using dynamic visualizations and KPI reporting


Implemented DAX calculations, data modeling, and interactive charts to transform raw banking data into meaningful business insights for financial analysis and decision-making.
